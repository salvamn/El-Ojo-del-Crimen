¿Qué es El Ojo del Crimen?

El Ojo del Crimen es una herramienta educativa y de auditoría de seguridad que demuestra las vulnerabilidades presentes en la gestión local de contraseñas de navegadores basados en Chromium (como Google Chrome, Microsoft Edge, Brave, entre otros) en sistemas Windows.

Esta aplicación permite extraer, descifrar y visualizar las credenciales almacenadas localmente en el equipo del usuario, usando técnicas que ilustran cómo un programa con permisos estándar puede acceder a datos sensibles.

Propósito

Esta herramienta está diseñada exclusivamente para fines educativos y de auditoría personal, con el objetivo de aumentar la conciencia sobre las vulnerabilidades que existen en la gestión local de credenciales y promover mejores prácticas de seguridad.

La herramienta localiza la base de datos SQLite donde los navegadores Chromium almacenan las credenciales encriptadas.

2️⃣

Extracción de Datos

Se accede a la tabla "logins" de la base de datos y se extraen los campos URL, usuario y contraseña encriptada.

4️⃣

Presentación

Las contraseñas desencriptadas se muestran en una interfaz organizada con opciones de búsqueda y exportación.

Preguntas Frecuentes

Resolvemos las dudas más comunes sobre la herramienta

¿Es legal usar esta herramienta?

+

Sí, es completamente legal cuando se usa en tu propio equipo para fines educativos o auditoría personal. Sin embargo, usar esta herramienta en equipos ajenos sin autorización explícita puede violar leyes de privacidad y acceso no autorizado.

¿Qué navegadores son compatibles?

+

La herramienta funciona con navegadores basados en Chromium como Google Chrome, Microsoft Edge, Brave, Opera, y otros que utilicen la misma estructura de almacenamiento de credenciales.

¿Cómo puedo protegerme de estas vulnerabilidades?

+

Recomendamos usar gestores de contraseñas dedicados como Bitwarden o 1Password, habilitar la autenticación de dos factores, y evitar guardar contraseñas sensibles en el navegador.

¿Puedo usarlo en sistemas que no sean Windows?

+

Actualmente, la herramienta está diseñada para sistemas Windows 10/11.
